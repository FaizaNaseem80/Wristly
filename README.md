## Installation

1. Clone the repository:

   ```bash
  git remote add origin https://github.com/FaizaNaseem80/Wristly.git
   ```
2. Navigate into the project directory:

   ```bash
   cd wristwatch-store
   ```
3. Install dependencies:

   ```bash
   npm install
   ```
4. Start the development server:

   ```bash
   npm start

   ```
5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

## Usage

* Navigate through the homepage to see available products.
